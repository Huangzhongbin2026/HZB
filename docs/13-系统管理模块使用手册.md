# 系统管理模块使用手册（产品视角）

## 1. 菜单配置
1. 进入“系统管理-菜单配置”。
2. 通过树形列表新增目录/菜单/按钮。
3. 配置菜单名称、图标、路由、排序、父级、权限标识。
4. 保存后可进行导出，用于环境迁移。

## 2. 数据字典
1. 创建字典分类（如 TASK_STATUS）。
2. 在分类下新增字典项（编码、名称、值、排序、状态）。
3. 业务表单下拉框可通过字典编码直接拉取。

## 3. 角色管理
1. 新建角色并设置所属部门与描述。
2. 勾选页面权限（菜单+按钮）。
3. 设置数据范围（全部/本部门/本人/自定义部门）。
4. 设置字段权限（可见/可编辑/隐藏）。
5. 通过权限预览确认后发布。

## 4. 用户管理
1. 新增用户并填写账号、手机号、飞书ID。
2. 分配一个或多个角色。
3. 支持重置密码、批量启用禁用。
4. 连续输错密码5次将锁定1小时。

## 5. 系统日志
1. 按操作人、时间、模块、IP、关键字组合查询。
2. 超级管理员可按时间范围清理日志。
3. 支持导出日志用于审计留档。
