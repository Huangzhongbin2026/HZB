# 本机环境安装指南

## 1. 必装软件
- Git
- Node.js 20+
- pnpm 8
- JDK 17
- Maven 3.9+
- Docker Desktop

## 2. 快速自检
在项目根目录执行：

```bash
powershell -File ./scripts/check-env.ps1
```

## 3. 本地数据库启动
```bash
cd deploy
docker compose -f docker-compose.local.yml up -d
docker compose -f docker-compose.local.yml ps
```

数据说明：
- MySQL 数据卷：deploy/mysql-data
- Redis 数据卷：deploy/redis-data
- 初始化脚本：docs/03-数据库初始化脚本.sql

## 4. 后端启动（无 Maven 时）
建议优先安装 Maven；若暂未安装，可先只做前端开发与接口联调模拟。

安装 Maven 后：
```bash
cd server/supply-task-service
mvn clean package -DskipTests
java -jar target/supply-task-service-1.0.0.jar --spring.profiles.active=dev
```

## 5. GitHub 推送失败排查
若出现无法连接 github.com:443：
- 检查公司网络是否限制外网。
- 配置 HTTP/HTTPS 代理。
- 改用 SSH 地址并配置 SSH Key。

示例：
```bash
git remote set-url origin git@github.com:Huangzhongbin2026/HZB.git
git push -u origin develop
git push origin --tags
```
