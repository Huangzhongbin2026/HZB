# HZB 代码备份仓库

这是一个用于代码备份的仓库，用于存储和管理个人或项目代码的备份。

## 目录结构

```
HZB/
├── README.md       # 仓库说明文档
├── projects/       # 各项目代码备份目录
└── scripts/        # 常用脚本备份目录
```

## 使用说明

1. 将需要备份的代码放入对应的目录中
2. 使用 `git add`、`git commit`、`git push` 命令将代码推送到远程仓库
3. 可按项目或时间命名子目录，方便管理

## 备份规范

- 每次备份时，建议在 commit message 中注明备份时间和内容简介
- 敏感信息（如密码、密钥等）请勿上传至仓库
