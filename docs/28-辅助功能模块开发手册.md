# 辅助功能模块开发手册（技术视角）

## 1. API 前缀
- `/api/v1/supply/system/aux/`

## 2. 统一要求
- 所有增删改、导入导出、开关变更接口添加 `@SysLog`。
- 人员关联字段需校验用户存在。
- 字典依赖字段需校验字典值有效。

## 3. 匹配接口
- `/leave-configs/match/by-user`：查询请假状态。
- `/virtual-products/match`：按产品型号返回自动回复。
- `/leave-agent-products/match`：按产品型号+原统筹返回代理统筹。
- `/area-coordinators/match`：按部门/省份/关键词匹配区域统筹。

## 4. 缓存策略
- 虚拟产品匹配结果存 Redis，key=`aux:virtual:{model}`。
- 失效策略：数据变更后主动删除缓存。

## 5. 飞书集成预留
- `FeishuMessageClient.sendCard()` 统一封装。
- 支持模板code、接收人、变量参数。
