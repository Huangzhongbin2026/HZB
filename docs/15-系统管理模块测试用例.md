# 系统管理模块测试用例

## 1. 菜单权限
1. 普通角色登录仅展示授权菜单。
2. 无按钮权限时按钮隐藏。
3. 菜单批量禁用后前端不再渲染。

## 2. 字典管理
1. 字典分类编码唯一校验。
2. 字典项编码唯一校验。
3. 字典项按关键字模糊搜索准确。

## 3. 角色与数据权限
1. 角色继承后权限完整继承。
2. 多角色叠加后权限为并集。
3. 数据范围分别验证 ALL/DEPT/SELF/CUSTOM_DEPT。
4. 字段级权限验证可见/可编辑/隐藏。

## 4. 用户安全
1. 飞书ID唯一校验。
2. 手机号唯一校验。
3. 连续输错5次锁定1小时。
4. 重置密码后旧密码失效。

## 5. 系统日志
1. 核心增删改操作均记录日志。
2. 日志支持按条件组合查询。
3. 仅超级管理员允许日志清理。
4. 日志导出文件格式正确。
