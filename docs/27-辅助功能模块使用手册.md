# 辅助功能模块使用手册（产品视角）

## 1. 请假配置
- 按人员ID、姓名、时间筛查。
- 新增/编辑时同一人员请假时间不可重叠。
- 可导入导出 Excel，导入前先下载模板。

## 2. 虚拟产品管理
- 维护产品型号与自动回复内容。
- 主任务命中同型号后自动回填回复内容。

## 3. 消息推送管理
- 配置消息名称、路由、飞书模板和开关。
- 开关关闭后立即停止该路由消息发送。

## 4. 请假代理产品型号管理
- 维护原统筹与代理统筹关系。
- 主任务遇请假统筹时自动替换为代理统筹。

## 5. 区域统筹划分管理
- 配置销售部门、省份、关键词与统筹人。
- 系统按“精准优先+关键词兜底”匹配统筹。
